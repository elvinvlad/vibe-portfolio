# Vlados Portfolio - Task Plan

## Protocol 0: Initialization
- [x] Set up `task_plan.md` and `findings.md`

## B - Blueprint (Vision & Logic)
- **Goal**: Create a premium SaaS-like structure but with an elite Dark Neon Biotech aesthetic for Vlados (Business Automation with AI).
- **Style**: Notion/Linear structure (clean typography and layouts) but wrapped in `#0A0A14` dark backgrounds, glowing purple accents (`#7B61FF`), and soft dark mode shadows.
- **Stack**: React + Tailwind CSS.

## L - Link (Connectivity)
- Initialize React + Vite + Tailwind CSS.
- Link Google Fonts (Unbounded, Noto Sans).

## A - Architect (The 3-Layer Build)
- `src/Architecture/` - Logic (React hooks, state management, animations)
- `src/Network/` - JSON data schemas (portfolio items, features, social proof)
- `src/Terminal/` - UI (React components, Tailwind classes)

## S - Stylize (Visual Identity)
- Strict Colors: #0A0A14, #10101E, #7B61FF, #9E88FF, #C4B5FD, #FFFFFF, #BBBBD0, #7B7B9A.
- Typography: Unbounded (Headings), Noto Sans (Body).
- Structural Rules: Golden Ratio Radius (`rounded-[2rem]`/`rounded-3xl`), Floating Island Nav, 50/50 splits.
- Effects: Glassmorphism, 0.05 opacity SVG noise overlay, fade-in on scroll.

## T - Trigger (The Soul)
- Hover lift effects on buttons and cards (`translate-y-[-2px]`).
- Smooth entrances (Framer Motion).
